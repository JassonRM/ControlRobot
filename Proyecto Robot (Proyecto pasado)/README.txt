Robot virtual
Jasson Rodríguez Méndez
Taller de programación

El proyecto fue programado haciendo uso de Python versión 3.6.1
Para la ejecución del robot virtual es requerido la instalación de las librerías externas:
PyGame
PyAudio
PortAudio
SpeechRecognition

El motor de reconocimiento de voz utilizado es el Google Speech Recognition API, por lo tanto no puedo asegurar el correcto funcionamiento de este a la hora de ejecutar el programa. Además, para su ejecución debe poseer una conexión a internet.
